# Instalation:

 *Linux
 ```
 apt install git
 
 git clone https://github.com/damirstash34/Url-Injection-Checker
 
 cd Url-Injection-Checker
 
 cd installs
 
 ./modules_install_linux.sh
```

 *Termux
```
  pkg install git
  
  git clone https://github.com/damirstash34/Url-Injection-Checker
  cd Url-Injection-Checker

  cd installs

  ./modules_install_linux.sh
```
 *Windows
  Install python
  Download .rar archive
  Run file UIC/installs/modules_install_windows.py
